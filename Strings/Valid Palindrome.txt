A phrase is a palindrome if, after converting all uppercase letters into lowercase letters and removing all non-alphanumeric characters"[^a-zA-Z0-9]", it reads the same forward and backward. Alphanumeric characters include letters and numbers.

class Solution {
    public boolean isPalindrome(String s) {
        String str = s.toLowerCase();
        str=str.replaceAll("[^a-zA-Z0-9]","");
        int low=0;
        int high=str.length()-1;
        while(low<high){
            if(str.charAt(low)!=str.charAt(high)) return false;
            low++;
            high--;
        }
        return true;
    }
}

TC - O(N)
SC - O(1)
